# PaymentStatus

The transaction status can be one of  ["SUCCESS", "NOT_ATTEMPTED", "FAILED", "USER_DROPPED", "VOID", "CANCELLED", "PENDING"]


## Values

| Name           | Value          |
| -------------- | -------------- |
| `Success`      | SUCCESS        |
| `NotAttempted` | NOT_ATTEMPTED  |
| `Failed`       | FAILED         |
| `UserDropped`  | USER_DROPPED   |
| `Void`         | VOID           |
| `Cancelled`    | CANCELLED      |
| `Pending`      | PENDING        |
